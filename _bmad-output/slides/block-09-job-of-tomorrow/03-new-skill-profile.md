# The evolving developer skill profile

**Skills that gain importance:**
- Prompt engineering as a professional skill — writing structured, verifiable specifications
- Security reasoning — knowing what AI cannot catch, applying judgment where it fails
- Evidence and traceability — structuring AI-assisted work so it is auditable and defensible
- Tool evaluation — matching tools to project constraints and comparing them critically

**Skills that remain irreplaceable:**
- System architecture and design reasoning
- Client and business context judgment
- Accountability — AI does not sign off on code. You do.
