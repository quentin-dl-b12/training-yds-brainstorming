# AI in the developer context — 2026

**What "AI" means here:** LLMs and specialized ML models integrated into everyday tooling. Not robots. Not AGI.

**What an LLM is in plain terms:**
- Trained on vast amounts of text and code
- Generates probabilistic completions — it *predicts*, it does not *know*
- Fast, fluent, confident — and occasionally completely wrong

**The key mental model:**
> A very fast, very well-read, very confident junior colleague who needs supervision.

Classic ML models → Generative LLMs → LLM-powered tools embedded in IDEs and CI/CD
