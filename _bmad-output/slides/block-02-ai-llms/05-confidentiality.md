# AI and confidentiality — the basics

| Tool type | Risk | Rule |
|---|---|---|
| Cloud LLMs (ChatGPT, Copilot, Claude) | ⚠️ Check terms | Code sent may be retained or used for training |
| Enterprise / tenant-isolated LLMs | 🔶 Safer | Still audit what you allow |
| Local LLMs (Ollama, LM Studio) | ✅ Nothing leaves the machine | The answer for restricted client projects |

**Never paste into any cloud AI tool:**
- Credentials or API keys
- Personally identifiable information (PII)
- Proprietary client algorithms or business logic
