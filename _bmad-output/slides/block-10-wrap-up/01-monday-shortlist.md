# Your Monday shortlist

1. **Install Copilot** (or enable Ollama) — use it on a real task, then review everything before committing
2. **Run OWASP Dependency-Check or Snyk** on one project you are currently working on
3. **Next security-sensitive feature:** write a one-page spec including security requirements *before* touching the code
4. **Audit your CI/CD:** find one place where a security gate is missing or disabled
5. **Next code review:** ask "What would an attacker do with this code?" — even without a tool

*Pick one. Start there.*
