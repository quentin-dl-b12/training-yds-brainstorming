# AI agents — what they are and why they matter

**An agent** = an LLM given tools, memory, and the ability to autonomously execute multi-step tasks: browse, write files, run commands, call APIs.

**In practice:**
- GitHub Copilot Workspace: plans and executes code changes across a repository
- Cursor Agent mode: edits multiple files from a single instruction
- Autonomous agents can run a security scan, read the output, propose fixes, and open a PR — with minimal human input

**The governance challenge:**
The more autonomous the agent, the more important human review and approval gates become.

*Not every team is ready for agents today — but understanding them is table stakes for 2026.*
