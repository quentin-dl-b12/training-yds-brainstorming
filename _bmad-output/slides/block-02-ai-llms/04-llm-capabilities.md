# What LLMs can — and cannot — do reliably

**CAN DO:**
- Generate code from a description
- Explain existing code and suggest fixes
- Spot common patterns and vulnerabilities
- Summarize documents, draft specs and documentation
- Run multi-step agent tasks

**CANNOT DO reliably:**
- Reason about novel or context-specific security threats
- Guarantee correctness of generated code
- Know your project context, architecture, or client constraints
- Replace human judgment

**The hallucination problem:**
LLMs produce plausible-sounding wrong answers.
This is a **security risk**, not an inconvenience.
