# Security is a team responsibility

In an ISO27001-certified company, security is not just the CISO's problem.

| Role | Security responsibility |
|---|---|
| **Developer** | Secure coding (A.8.28), secrets hygiene, dependency hygiene |
| **Code reviewer** | Verify security intent, not just functionality |
| **DevOps / CI** | Security gates in the pipeline, environment hardening (A.8.31) |
| **Manager / lead** | Enforce the process, allocate time for security testing (A.8.29) |

Everyone contributes. The CISO defines the policy. The team delivers it.

> "Security is a quality attribute, like performance.
> You would not ship code that crashes on every request."
